## Advanced-Data-Structures

<p style="text-align: justify;">
This repository contains the materials for the Advanced Data Structures course offered in the Ph.D. program. The course lecture notes, assignments, and the final project are provided in this repository. The final project was submitted to the International Conference on Computer and Knowledge Engineering (ICCKE 2022). Although the paper was not accepted for publication, the project and its associated materials are included here for educational and research purposes.
